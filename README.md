# このパッケージについて
recursive-input.styは、指定されたフォルダ内にある全てのファイルを再帰的に探索し、\inputするパッケージです。
\inputの順番は2015/08/17の時点でOS依存となっています。また、ファイル・フォルダの除外指定は現在実装中です。
